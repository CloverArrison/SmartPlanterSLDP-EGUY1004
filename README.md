# JAR Plant Solutions - Smart Planter - SLDP EG-UY 1004

## Made by: Clover Arrison (Project Lead, Software), Riad Mahamud Rahat(Electronics), Andrew Jiang (3D Design)

Our proposed prototype is a smart planter designed to improve plant care by automatically managing key environmental conditions. The device will include a plant container with integrated sensors: a moisture sensor to detect the soil’s moisture level, a temperature sensor to monitor temperature, a light sensor, and a ventilation system to maintain proper airflow. The plant will be encapsulated with a transparent shield (transparent plastic dome) for proper light flow and protection, which will be attached to the plant container.  At the top of the encapsulated shield, there will be a fan for the proper airflow. The device's main function is to regulate watering, temperature, and ventilation so plants can survive longer in both retail environments and homes.
